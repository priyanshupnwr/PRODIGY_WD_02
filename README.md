# PRODIGY_WD_02 - Stopwatch Web Application

## Overview

This project is submitted as **Task 02** for the **Prodigy InfoTech Web Development Internship**.

The objective of this task was to build a responsive stopwatch application using HTML, CSS, and JavaScript with accurate time tracking and user interaction.

---

## Features

- Start Stopwatch
- Pause & Resume
- Reset Stopwatch
- Lap Recording
- Clear Lap Records
- Millisecond Precision
- Responsive Design
- Clean and User-Friendly Interface

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
